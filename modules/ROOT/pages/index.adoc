= Preface
Stefan Wintermeyer <sw@wintermeyer-consulting.de>

The book you are about to read targets **Ruby 4.0** and
**Ruby on Rails 8.1**. It's written the way I'd teach these in a
hands-on workshop: clean, short examples, one concept at a time,
skip-friendly. Every example has been re-run against these
versions; screenshots come from fresh Rails 8 scaffolds. Chapters
added for the 5.2 → 8.1 refresh cover Hotwire (Turbo + Stimulus),
the built-in authentication generator, and the Solid trifecta
(Solid Queue / Solid Cache / Solid Cable).

I write for developers who learn best by following clean, short
examples. Rather than building one big application throughout,
I prefer smaller stand-alone code — so you can skip pages or
whole chapters without losing context. On the other hand, I will
ask you to spin up a new Rails application more often than you
might expect. Word of warning: I'm not going to sprinkle CSS
beauty anywhere. ;-)

The sidebar lets you jump to any chapter at any time — nothing is
locked. That said, the book is written to be read front to back:
examples build on earlier ones, and the fastest path from zero to
a working Rails app is still "start at the top and keep going".
