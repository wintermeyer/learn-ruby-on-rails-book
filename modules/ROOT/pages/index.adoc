= Welcome
Stefan Wintermeyer <sw@wintermeyer-consulting.de>

The book you are about to read targets **Ruby 4.0** and
**Ruby on Rails 8.1**. Every example has been re-run against these
versions; screenshots come from fresh Rails 8 scaffolds. Chapters
added for the 5.2 → 8.1 refresh cover Hotwire (Turbo + Stimulus),
the built-in authentication generator, and the Solid trifecta
(Solid Queue / Solid Cache / Solid Cable).

Pick a chapter from the sidebar. I recommend reading in order, but
each chapter stands on its own so you can skip ahead if a topic
doesn't interest you.

== Don't be a stranger

A book like this gets better every time a reader sends me a note.

* Did a chapter click, or did it leave you more confused than
  before? Either answer is useful.
* Found a typo, a broken example, code that no longer works on
  the current Rails or Ruby version, a link that 404s?
* Missing a topic you expected? Or curious about something this
  book doesn't cover?

Write me at mailto:sw@wintermeyer-consulting.de[sw@wintermeyer-consulting.de].
Feedback, bug reports, kind words, angry words — all welcome.
